A complete Full Stack solution developed within the Salesforce ecosystem, bridging a high-conversion Landing Page (Front-end) with robust data capture and an administrative management panel (Back-end).

Key Features
- LWC Landing Page: Modern, responsive, and conversion-optimized interface.
- Real-time Integration: Direct Apex Controller connection for instant Lead insertion.
- Traffic Analytics: Intelligent system that tracks site visits and automatically calculates the Conversion Rate.
- Real User Filtering: Apex logic that excludes logged-in employees from metrics, ensuring data reflects only external visitor (Guest User) behavior.
- Custom Toast System: Visual feedback notifications designed specifically for public-facing LWR sites.
- Lead Manager Dashboard: Internal administrative panel for KPI visualization and real-time Lead monitoring.

Tech Stack
- Salesforce: LWC, Apex, Experience Cloud (LWR), SOQL.
- Web: HTML5, CSS3, JavaScript (ES6+).
